# Robust-Parallel-Batch-Scheduling-Decision-Dependent-Uncertainty 

The instance generation code and solver implementation of the research project. 

The instance_io.py can be used to generate instances. 

The deterministic_solver.py is the implementation of the arc-flow formulation by Zheng et al. (2025). 

The robus_solver.py is the implementation of the robust bilevel formulation using dual embedding.
